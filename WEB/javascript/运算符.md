## 3.1 赋值运算符

3.1.1 简单赋值
   var a = 10;

3.1.2 复合赋值
  如果在等号前面加上+-*/，就可以完成复合赋值。 =+ -= *= /= <<= 
       var a = 10;
        a = 10+1;
       //简写成 a+ = 1

3.3 关系运算符

###3.4 相等运算符 
 == === ！= ！== 返回的都是布尔值 相等和不相等 都是西安转换数据类型再进行比较 全等和不全等 金比较不转换数据类型
需要注意的是 ==不会比较数据类型 ===会比较数据类型

###3.6 逻辑运算符
再运用逻辑运算符之前。 需要掌握数据类型的知识，因为 说有逻辑操作符返回值都是布尔值 一次 返回值转换非常重要

3.61 逻辑非
逻辑非，首先会将！ 后面的值转换为一个布尔值，然后再对其求相反

3.62 逻辑与&
&左右两边只要有一个是false就是false，两边都是true才是true；

3.63 逻辑或 || 左右两边只要有一个是true就是true 值有两边都是false才是false



3.64 短路与&&
 它的与那算过程更加高效 ，只要左边结果是false那么后面就不运算了
